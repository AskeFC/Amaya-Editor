wxWidgets installation

  1. Download wxMSW-$version.zip form wxWidgets Web site.
  2. Unzip wxMSW-$version.zip into C:\src\ and rename wxWindows-$version to wxWidgets
  3. Copy C:\src\Amaya\WindowsWX\wx-setup\setup.h to C:\src\wxWidgets\include\wx\msw
  4. Open the workspace C:\src\wxWidgets\build\msw\wx.dsw
  5. Run : Build -> Batch Build -> Rebuild All
  6. Open the workspace C:\src\wxWidgets\contrib\build\xrc\xrc.dsw
  7. Run : Build -> Batch Build -> Rebuild All

AmayaWX compilation

  1. Open the workspace C:\src\Amaya\WindowsWX\amaya.dsw
  2. Press F7 to compile Amaya
