This manual presents the user interface of Amaya. The user interface on your computer could be a little different, depending on your system and its configuration. This is not a complete user manual. Its purpose is only to help users to discover the main features of the software.

This document is organized as a book and you can use the Tools/Make Book command to build and print the whole manual.

Many commands in Amaya are the same as those provided by common word processors and editors. They are not presented here. Only specific aspects are described.
