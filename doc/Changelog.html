CVS Amaya changes

This file lists bug fixes and new features introduced in the CVS version of Amaya since the last release (Amaya 11.2)

New Features

  • The https protocol is now supported
  • New version of the template editor: creation of XTiger libraries, components, imports, etc.
  • Automatic open of the Structure view for template instances
  • The Windows install script adds a "Edit with Amaya" entry for the files Amaya can handle
  • The vertical split opens the Structure view and the horizontal split opens the Source view

Bug fixes

  • The link dialogue didn't allow one to insert relative links into a remote document
  • Fix some rendering bugs
  • Improve the edit of template instances: management of options and attributes
  • Optimize the refresh of tool panels
  • PNG images were not displayed on 64-bit platforms
