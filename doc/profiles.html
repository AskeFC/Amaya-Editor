Profiles in Amaya

How to use it :

The definition file thot.rc contains two new fields :

  • Profiles_File :

    This field contains the location of the file which defines the available profiles.

    e.g : Profiles_File=/home/bonameau/.amaya/Profiles.

    You have to create this file and fill it in before using profiles.

    This file looks like this :

    [EditorNovice] /* Profile's name */

    OpenDoc /* List of functions added in the profile */

    OpenDocInNewWindow

    New

    NewTemplate

    NewCss

    SaveDocument

    SaveDocumentAs

    MenuAttribute

    ConfigProfile

    .....

    [EditorMaster] /* Profile's name */

    OpenDoc /* List of functions added in the profile */

    ....

    New

    NewTemplate

    NewCss

    ....

    MenuAttribute

    .....

    ConfigProfile

    ...

    [Browser] /* Profile's name */

    OpenDoc /* List of functions added in the profile */

    OpenDocInNewWindow

    Reload

    GotoPreviousHTML

    GotoNextHTML

    SetupAndPrint

    PrintAs

    TtcCloseDocument

    AmayaClose

    TtcCloseView

    ....

    HelpBrowsing

    [...........]

    .....

  • Profile :

    This field contains the current profile to use.

    e.g : Profile=[Browser]

Use the Profile entry from the config menu to set these fields.

After having done these modifications, just restart Amaya with the new profile.

Obviously, if theses fields are empty, Amaya will start with the full interface.

Notice : The definition profile will later use a list of modules instead of a list of functions.
