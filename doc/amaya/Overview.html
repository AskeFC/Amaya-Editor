User Documentation and Design Overview

The documentation provided with this release is not complete yet, but it should be enough to understand how to use and configure Amaya. Some documentation is also available for developers.

Using Amaya

  • Short presentation of the user manual for editing and browsing HTML pages.
  • Description of Amaya and of its features in the W3C Note "An Introduction to Amaya".
  • Description of the implementation of Cascading Style Sheets, with an overview of their user interface.
  • Note on using proxies with Amaya.
  • Description of the keyboard control and ISO-Latin-1 characters support.

Compiling Amaya

For those who want to port Amaya to a new platform or adapt it to their needs, there is a document which explains how to compile the Amaya environment for Unix platforms from the distributed source tree.

The Amaya compilation differs on PC/Windows platforms. See Compiling Amaya On Windows NT for more information on how to compile and install Amaya.

Developer's corner

Those who want to extend or change the Amaya application need to read following documentation:

  • Overview of the Amaya architecture.
  • The Thot API manual
  • The Thot Application generation manual
  • The Thot language manual
