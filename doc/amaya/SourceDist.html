Amaya Alpha 0.95a Source Release

About this release

Release date: January 22, 1977

This is the first source release. It is reserved to W3C members for one month.

Amaya is a comprehensive client environment for testing and evaluating new proposals for Web standards and formats.

This release should be considered as a snapshot of a work in progress. A large part of the intended features of Amaya are implemented, but some of them are not complete yet and the software is not supposed to be very robust. The user interface may change and can be improved.

Check out the list of new features.

Getting the source distribution

Amaya is designed as an application on top of the Thot editing tool kit. Therefore, the source release is constituted of two tar files that you must download first:

  • Download the amaya-src-0.95a.tar.gz Amaya source code distribution (HTTP) (FTP)
  • Download the Thot source code (FTP)

With these files, you can compile Amaya (follow the directions given in the building note). If you are using one of the supported platforms, please download the libraries that have been precompiled for you:

  • Linux ELF format

    Download amaya-libs-LINUX-ELF-0.95a.tar.gz (HTTP) (FTP)

    and thot-libs-LINUX-ELF-2.0a.tar.gz (FTP)

  • Linux a.out format

    Download amaya-libs-LINUX-0.95a.tar.gz (HTTP) (FTP)

    and thot-libs-LINUX-2.0a.tar.gz (FTP)

  • Sun Solaris 2.5

    Download amaya-libs-solaris2-0.95a.tar.gz (HTTP) (FTP)

    and thot-libs-solaris2-2.0a.tar.gz (FTP)

  • SunOs 4.1.x

    Download amaya-libs-SUN4-0.95a.tar.gz (HTTP) (FTP)

    and thot-libs-SUN4-2.0a.tar.gz (FTP)

  • SGI Irix

    Download amaya-libs-SGI-0.95a.tar.gz (HTTP) (FTP)

    and thot-libs-SGI-2.0a.tar.gz (FTP)

  • IBM AIX 4.1

    Download amaya-libs-RS6000-0.95a.tar.gz (HTTP) (FTP)

    and thot-libs-RS6000-2.0a.tar.gz (FTP)

Amaya includes a spell-checker. If you wish to use this feature, you must download and install the dictionaries contained in the following files:

  • Amaya dictionaries (same files for all architectures)

    Download the English dictionary

    Download the French dictionary

Installing Amaya

Please follow the directions given in the building note.

If you want to port Amaya on a platform that is not supported yet, read the porting section.

Further documentation is provided inside the distribution tar file and in the Amaya on-line documentation page.
