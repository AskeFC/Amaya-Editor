Amaya 3.0 Source Release

About this release

Release date: 31 March 2000

Amaya is intended to be a comprehensive client environment for testing and evaluating new proposals for Web standards and formats. A large part of the intended features of Amaya are implemented in this release, but some of them are not complete yet. Amaya is designed as an application on top of the Thot editing tool kit.

Check out the list of new features.

Getting the source distribution

The complete Amaya source distribution for Windows and Unix platforms is available at:

  • amaya-src-3.0.tgz (about 4,333,000 bytes) Amaya source code distribution (http) (ftp-us) (ftp mirror in Australia)

Checkout from CVS

Checking out the code base directly from our CVS repository allows you to follow the development of Amaya closely and to contribute much easier. Also, you will get a lot of bug fixes that are not in the distribution tar ball. In fact, if you intend to provide patches then you should use the CVS repository! Access to the CVS base is only available in read-only mode.

The tar file snapshot of the CVS tree is being made at daily intervals. It is available at ftp://ftp.w3.org/pub/amaya/amaya-snapshot.tar.gz. That tar file is not equivalent to the standard source distribution. For that tar file one has first to generate include files and the file configure does process as explained in the section " Getting libwww Boostrapped" .

Installing and compiling Amaya sources

Please see the information about the Amaya sources. Make sure you're using one of the supported platforms. If you want to compile Amaya:

  • on a Unix platform, read the Unix compiling section.
  • on PC/Windows platform read the Windows compiling section.

Installing dictionaries

Amaya includes a spell-checker. If you wish to use this feature, you must download and install a dictionary for the language you want to check. These dictionaries are not complete, so the spell-checker could report spelling errors just because the dictionary doesn't include these words.

This is the list of dictionary tar files currently available. :
