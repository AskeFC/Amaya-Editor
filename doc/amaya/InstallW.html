How to install the Windows Amaya Binary Release

The Amaya installation process is automatic. You have to doubleclick on the amaya setup program and follow the instructions of the setup process. A short cut is automatically added to the Windows start menu and you can use your own short cut by creating it from the amaya icon generated by the setup program.

If you choose to install amaya in the directory X (example: C:\Program Files), the setup program will create a new directory called X:\Thot populated with a few files and directories. The amaya.exe file is located in the directory X:\Thot\Windows\bin.

There are two different Amaya versions: one without dictionary and one including French and English dictionaries. Both unclude online documentation.
