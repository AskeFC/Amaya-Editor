How to install the Windows Amaya Binary Release

Choose a place for Amaya

One must choose a place to install it. We suggest c:\ to load the binary distribution for Windows: amaya-WindowsPlatform-Release.ZIP

Extract the distribution

Move the distribution to this directory and extract the distribution.

unzip amaya-WindowsPlatform-Release.ZIP

The unzip should create a new directory called Thot populated with a few files and directories. Dictionaries and online documentation are archived in gzipped tar files:

  • English dictionary English.tar.gz (660,461 bytes) gzipped tar file.

    download (http us), (ftp us), (ftp eu).

  • French dictionary French.tar.gz (801,777 bytes) gzipped tar file.

    download (http us), (ftp us), (ftp eu).

  • Online documentation amaya-user-doc.tar.gz gzipped tar file.download (http us), (ftp us), (ftp eu).

We recommand to download the winzip tool which recognizes .gz and .zip files.

Set up the environment and start Amaya

Platform could be NT or 95. If you are using Windows NT, the application is located in Thot/Winnt/bin/ and if you are using Windows'95 the application is located in Thot/Win95/bin/
