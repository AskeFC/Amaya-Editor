How to install the Windows Amaya Binary Release

The Amaya installation process is automatic. You have to doubleclick on the amaya setup program and follow the instructions of the setup process. A short cut is automatically added to theWindows start menu and you can use your own short cut by creating it from the amaya icon generated by the setup program.

If you choose to install amaya in the directory X (example: C:\Program Files), the setup program will create a new directory called X:\Thot populated with a few files and directories. The amaya.exe file is located in the directory X:\Thot\Windows\bin.

Dictionaries and online documentation are archived in gzipped tar files:
