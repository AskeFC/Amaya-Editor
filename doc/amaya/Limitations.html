Known limitations in the Amaya Alpha 1.0 release

The current version of Amaya has several limitations, which will be removed in the next releases.

Undo
There is no Undo command. Be careful when deleting some part of a document.
Printing
Style defined by style sheets is currently ignored when printing a document.
Navigation
The Forward and Backward commands (buttons) do not always work properly in thisversion.
Protocols
Only HTTP is supported. URLs starting with mailto, news, file, ftp, etc. cannot be activated.
