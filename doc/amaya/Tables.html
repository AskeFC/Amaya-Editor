Editing tables

There is no separate table editor in Amaya. A table is simply a structure described by the HTML DTD and it is edited as such. The same commands apply to tables as to other HTML structures.

Creating a new table

When the selection is empty and the user actives the table button or selects the Table entry from the Types menu, Amaya display a form which allows the user to give the intial number of rows and columns of the new created table and its border value. if the user Cancel this form, the table is not created, otherwise the new table is created according to the user's choice. The space above the table is reserved for the caption. This caption is automatically selected after the creation and can be immediately deleted if the user wants to create a table without caption.

Amaya have had automatically generated empty cells. You can click on any of them and insert text, images, etc. you can also move the insert point from cell to cell by using arrow keys.

Adding a new column

New columns can be created very efficiently with the Return key. Move the insertion point to the end of the beginning of a cell (for example at the end of the Cell 1.1). By hitting the Return key twice (the first hit creates a paragraph in the first cell) you create a new cell and its corresponding column.

Adding a new row, new tbody

To create a new row, press Return three times when the end of the Cell 1.3 is selected. If you want to process more quickly, you can select the row by hitting the ESC key and press Return.

To create a new tbody, press Return four times (or three times if the cell is empty) and one times more to create a new paragraph after the table.

Removing a column

There is no specific command to create a new column. When you create a new cell, Amaya generates a new column. When you delete a cell within a column that becomes empty, Amaya deletes the column automatically. But there is a specific command to delete a column. This command is located in the submenu Table of the menu Types.

Note that the table model implemented in Amaya is the HTML 4.0 model. This includes features such as multiple bodies (tbody), header (thead) and footer (tfoot).
