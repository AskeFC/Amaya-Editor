Amaya Plan

The current development phase of Amaya is divided into several activities:

  • Unix and Windows usability
  • New functionalities on HTML, XHTML, MathML, SVG, XML
  • CSS support (editing/printing)
  • I18N support
  • GTK interface

Milestones/Commitments

Plan for the next release :

  • Planned date of the next release: end of April 2000
  • CSS: support of specificity and !important.
  • Improvement and bug fixes

List of possible future work (see also WishList from the Team) :

* I18N (in progress):

  • Specification draft "How to support Unicode in Amaya"
  • support of multibyte characters
  • experimentation with Unicode fonts
  • support of different writing directions

* CSS (in progress):

  • Specific command to help CSS editing.
  • extending the P language to stick with the CSS model: padding, margins, flowing, etc.

* XML:

  • parsing any XML document.
  • display XML document using CSS rules.

* DOM:

  • adding mouse events
  • adapting to the DOM API
  • experiments
