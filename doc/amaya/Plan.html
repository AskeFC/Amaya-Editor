Issues

List of possible future works:

* I18N:

  • support of multibyte characters
  • experimentation with Unicode fonts

  • support of different writing directions

* CSS:

  • changing the process which transforms CSS rules into Thot P language (the current process transforms the original CSS rules)
  • extending the P language to stick with the CSS model: padding, margins, flowing, etc.

  • CSS2 extensions

* HTML4:

  • forms
  • tables

  • frames(?)

  • see I18N

* DOM:

  • adding mouse events

  • adapting to the DOM API

  • experiments

* PICS:

  • integration of the PICS library
  • experimentation
