Editing mathematics

Amaya allows you to include mathematical expressions in your web pages, following the MathML (Mathematical Markup Language) specification.

In the current version of Amaya only a (small) part of MathML is implemented, actually a subset of the Presentation Markup. The purpose of this version is just to show how MathML may improve scientific communication over the Web. A more complete and more robust implementation is expected in future releases.

Creating a math expression

To create a math expression in a document, move the caret to the position where you want to insert that expression and click the Math button. It displays a menu that allows you to choose the first construct to be inserted. The available constructs are, from top to bottom:

  • Root, with an index (MROOT in MathML): x + 1 3
  • Square root (MSQRT): x + 1
  • Fraction (MFRAC): 1 x + 1
  • Expression with subscript and superscript (MSUBSUP): x i + 1 n
  • Expression with a superscript (MSUP): x n
  • Expression with a subscript (MSUB): x i
  • Expression with under- and overscript (MUNDEROVER): ∑ i = 1 n
  • Expression with overscript (MOVER): x → maps to y
  • Expression with underscript (MUNDER): x -
  • Expression with parentheses (MF): ( a + b )
  • A number (MN), an operator (MO), an identifier (MI), and some text (MTEXT).

Math_Structure view

Whenever a document contains some MathML elements, opening the Structure view opens actually two views:

  • the Structure view shows the main structure of the HTML document. Only the MATH or MATHDISP elements are shown, not their contents.
  • the Math_Structure wiew shows only MathML elements, but it shows all the details.

Editing math elements

To edit math elements, just move the insertion point to the desired position, with the mouse or the arrow keys, and type or select a construct from the math menu.

You don't need to enter each element with the math menu. When you type a character string in a MathML element, Amaya parses that string and generates the MO, MN, MI, MTEXT elements automatically. As an example, to enter x = 2 a + b just type x=2a+b.

The copy, cut and paste operations from the Edit menu are available for math and function as expected.
