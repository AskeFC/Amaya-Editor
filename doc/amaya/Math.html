Editing mathematics

Amaya allows you to include mathematical expressions in web pages, following the Mathematical Markup Language (MathML) specification. Mathematical expressions are handled as structured components, exactly in the same way as HTML structures. Therefore, you can manipulate math expressions in the same way you manipulate the rest of HTML documents. All editing commands are available for math, and there are some additional controls to enter mathematical constructs.

In the current version of Amaya only a (small) part of MathML is implemented, i.e. a subset of the Presentation Markup. The purpose of this version is to show how MathML may improve scientific communication over the Web. A more complete and more robust implementation is planed for future releases.

Creating a math expression

To create a math expression in a document, move the insertion point to the position where you want to insert that expression, and click the Math button (the rightmost button). It displays a menu that allows you to choose the construct to be inserted. In this version, the available constructs are, from top to bottom:

  • Root with index, MROOT in MathML: x + 1 3
  • Square root, MSQRT: x + 1
  • Fraction, MFRAC: 1 x + 1
  • Expression with subscript and superscript, MSUBSUP: x i + 1 n
  • Expression with a superscript, MSUP: x n
  • Expression with a subscript, MSUB: x i
  • Expression with under- and overscript, MUNDEROVER: ∑ i = 1 n
  • Expression with overscript, MOVER: x → maps to y
  • Expression with underscript, MUNDER: x -
  • Expression within parentheses, MF: ( a + b )
  • A number (MN), an operator (MO), an identifier (MI), some text (MTEXT).

Math_Structure view

Whenever a document contains some MathML elements, choosing the Show Structure item from the Views menu actually opens two views:

  • the Structure view shows the main structure of the HTML document. Only the MATH or MATHDISP elements are shown, not their contents.
  • the Math_Structure view shows only MathML elements, but it shows all their components.

You can edit math expressions either in the main view or in the Math_Structure view.

Editing math elements

To edit math elements, just move the insertion point to the desired position, with the mouse or the arrow keys, and type some character or select a construct from the Math menu.

You don't need to enter each element with the math menu. When you type a character string in a MathML element, Amaya parses that string and generates the MO, MN, MI, MTEXT elements automatically. For instance, to enter

x = 2 a + b

just type x=2a+b.
