Printing documents

Amaya is able to print your documents with or without CSS style sheets on both Unix and Windows platforms. On Unix platforms it is only able to generate Postcript description, so you need to have a PostScript printer or use a application able to translate the PostScript into your specific printer language.

There are two commands for printing a document:

  • The Setup and Print command (File/Setup and Print) allows you to set some parameters before printing.

    On Unix platforms these parameters are:

    • the paper format: A4. US
    • the orientation: Portrait, Landscape
    • the number of pages printed on the same sheet of paper: 1, 2, 4 pages per sheet
    • the output: file, or printer
    • options:
      • Manual feed
      • Generation of the table of cotents of the document
      • Generation of the table of links: a number is generated after each link in the document that refers an entry within the table of links; each table entry lists the corresponding URL
      • Printing of the document URL
      • Printing with CSS style sheets

    On Windows platforms, a first dialogue box propose following options:

    • Generation of the table of cotents of the document
    • Generation of the table of links of the document
    • Printing of the document URL,
    • Printing with CSS style sheets

    then the standard Windows dialogue will let you choose the other parameters.

  • The Print command (File/Print) prints the current document with default parameters or previous values set by the command Setup and Print. You may pay attention that parameters (Manual feed, Orinetation, etc.) are reset to the default value when the document change.

    On Windows platforms, the first times you launch the command Print the standard Windows dialogue will be displayed to let you check that the default printer is correct. Following printing will be launched without any dialogue.

In the button bar, the button gives you access to the command Print. Due to a trouble with the use of DLLs on Windows 95, that button calls the command Setup and Print on all Windows platforms.
