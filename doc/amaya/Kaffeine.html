Amaya and Kaffe

This document gives information on Amaya and Kaffe integration. Kaffe is a freely available Java interpreter with on the fly translation to native code (Just In Time aka JIT) on some platforms.

The Kaffe interpreter is statically linked in the Amaya binary, so there is no need to wonder about shared library dependencies.
