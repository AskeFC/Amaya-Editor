Amaya-Java Beta 1.1c Binary Release

About this release

Release date: November 28, 1997.

Amaya-Java is intended to be a testbed client environment for testing and evaluating new proposals for Web standards and formats using the existing Amaya framework and the Java language. Here is the documentation for the Java API and code available within the Amaya-Java framework.

The Amaya-Java binary distribution is available for Linux/i386, Linux/Sparc and Sun/Solaris. Users having other architectures are expected to compile the Amaya-Java source code.
