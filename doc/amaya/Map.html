Using Amaya

Client-side Image Maps

Using client-side image maps

Amaya allows you to use client-side image maps.

  • For activating an area in an existing map, double click the area with the left mouse button. The URL attached to this area is thenloaded.
  • For displaying existing areas, select item Show Map Areas in the Views menu.

Creating client-side Image Maps

For creating a new image map, first create the image by selecting the Image item in the Types menu or click button This prompts you with a dialogue box, for entering the file name of the image.

For creating an area, select item Rectangle, Circle, Polygon or Default in the AreaMap submenu in the Types menu when the image or an area is selected.

  • For creating a rectangle or a circle area, you must first select the top left corner by depressing the left button. Keep the button pressed and move to the position of the opposite corner, then release the button. The area is defined.
  • For a Polygon, select each point by clicking the corresponding position with the left button of the mouse. For the last point, click with the middle button.

When the shape of the area has been defined, you must enter the target of the link associated with that area. Just click the target or enter its URL if it is not displayed, in the same way you create other types of links.
