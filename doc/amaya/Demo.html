Starting Amaya

To start Amaya type:

amaya [-display host] [document]

Parameter [-display host] is optional. It defines the remote host screen where Amaya has to display Web pages. If this parameter is omitted, Amaya displays on the local host screen.

Parameter [document] is optional. It is the file name or the URL of the first document Amaya should display when starting. If no parameter is given, a default document is presented.
