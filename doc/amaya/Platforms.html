Current Amaya Platforms

This document lists the platfoms on which Amaya has been sucessfully compiled and the corresponding Target names. The Amaya Team development environment is based on Sun-Solaris and PC-Linux machines, so answering specific bug reports for other targets may be difficult.

  • Sun worstations running Solaris 2.5 : solaris2 .
  • Sun worstations running SunOS 4.1.x : SUN4 .
  • PC running ELF versions of Linux : LINUX-ELF .
  • PC running a.out versions of Linux : LINUX
  • Sun workstations running Linux : LINUX-SPARC
  • SGI worstation running Irix : SGI .
  • IBM workstations running AIX 4.1 : RS6000 .

Amaya is currently ported to the Windows 32 bit API, but this is work in progress.

Adding support for a new achitecture is explained in the porting section.
