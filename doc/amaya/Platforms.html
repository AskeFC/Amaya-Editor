Current Amaya Platforms

This document is an attempt to lists the platfoms on which Amaya has been sucessfully compiled and the corresponding Target names. The Amaya Team development environment is based on Sun-Solaris and PC-Linux machines, so answering specific bug reports for other targets may be difficult.

The table below lists the Platform, the Target nickname, whether amaya is known to work using the libWWW, or using Kaffe for Java network accesses. The support for Netscape Plug-ins is also available for a few platforms in the libWWW implementation. The last collumn indicate whether the Amaya Team provided a binary distribution for this platfom.
