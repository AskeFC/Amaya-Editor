Amaya-Java Beta 1.1c Source Release

About this release

Release date: November 28, 1997

Amaya-Java is intended to be a testbed client environment for testing and evaluating new proposals for Web standards and formats using the existing Amaya framework and the Java language.

The Amaya-Java source distribution is known to compile and run on Linux/i386 and Linux/Sparc. Users having other architecture will probably have more work to get Amaya-Java running than a simple recompilation from sources.

Getting the source distribution

Amaya is designed as an application on top of the Thot editing tool kit. Previously, users had to download two tar files, one for Thot and another for Amaya. The Amaya-Java distribution is now given in a single tar file:

  • Download the amaya-java-src-1.1c.tar.gz (about 13,807,000 bytes) Amaya source code distribution (HTTP US) (FTP US) (FTP EU).

To compile Amaya-Java, follow the directions given in the building note.

Compiling and installing Amaya

Please follow the directions given in the building note.

Further documentation is provided inside the distribution tar file and in the Amaya on-line documentation page.
