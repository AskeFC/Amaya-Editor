Compiling Amaya On Windows

This release of Amaya running on Windows NT and 95 is developed with MSVC++ (6.0). Makefiles are given with the source tar file.

Building Amaya using *.mak files:

  1. Load the Compilers makefile (Amaya\Windows\Compilers\Compilers.mak) and build it.
  2. Execute Compilers program, you will have a Window with a menu "File". Choose "File/Open" and load the file Amaya\Windows\amaya\amaya.mkf.
  3. Choose "File/Build", this will execute amaya compilers and generate files needed to build Amaya.
  4. Close your Compilers workspace and load Amaya\Windows\amaya.mak.
  5. Build Amaya.

Building Amaya using *.dsp files:

  1. Load the Compilers project (Amaya\Windows\Compilers\Compilers.dsp).
  2. Update the follwing dependencies (Menu Project/Dependencies):
    • Compilers depends on: LibThotKernel, app, prs, str and tra.
    • app depends on LibThotKernel and cpp
    • prs depends on LibThotKernel and cpp
    • str depends on LibThotKernel and cpp
    • tra depends on LibThotKernel and cpp
  3. Execute Compilers program, you will have a Window with a menu "File". Choose "File/Open" and load the file Amaya\Windows\amaya\amaya.mkf.
  4. Choose "File/Build", this will execute amaya compilers and generate files needed to build Amaya.
  5. Close your Compilers workspace and load Amaya\Windows\amaya.dsp.
  6. Update the following dependencies (Menu Project/Dependencies):
    • amaya depends on libThotEditor, libwww and thotprinter.
    • libpng depends on zlib.
    • libThotEditor depends on libjpeg and libpng.
    • libwww depends on zlib.
    • printlib depends on libjpeg, libpng and libThotTable.
    • thotprinter depends on printlib.
  7. Build Amaya.
