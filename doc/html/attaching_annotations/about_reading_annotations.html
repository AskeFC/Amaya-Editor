About Reading Annotations

Amaya identifies annotations with pencil annotation icons that are visually embedded in the document. If you single-click an annotation icon, you highlight the annotated text. If you double-click the icon, a separate window opens, displaying the annotation text and other metadata.

Below is an example of a pencil annotation icon embedded in the text, with the text highlighted.

An annotation has many properties including:

  • Physical location: Whether the annotation is stored on a local file system or on an annotation server.

  • Scope: Whether the annotation is associated with an entire document or just a fragment of the document.

  • Annotation type: The kind of annotation you can expect, for example, Comment or Query.
