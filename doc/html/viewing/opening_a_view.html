Opening a View

Amaya can present different views of a document simultaneously. Each view is displayed in a different window. This allows you to clearly see what is happening when you edit complex documents. It is also an aid for moving across a document.

Note: View options are only available from the normal browser view (formatted view).

To open a view:

  • Open the Views menu, and select the appropriate Show menu item:

    Show Menu Item

    Keyboard Shortcut

    View Strucure

    Alt+v  Alt+s

    View Alternate

    Alt+v Alt+I

    View Links

    Alt+v Alt+l

    View Table of Contents

    Alt+v Alt+c

    View Source

    Alt+v Alt+h

     

    Note:    The Show commands from the Views menu can also be used to raise the corresponding window when it is in the background.
