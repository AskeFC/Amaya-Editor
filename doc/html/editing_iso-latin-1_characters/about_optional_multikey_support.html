About Optional Multi-key Support for Motif and Windows

Amaya provides an optional multi-key input method based on a two-key sequence beginning with the accent and followed by the letter to be accented. For example, to obtain an a-tilde (ã), you first press ~ then type a. To obtain a tilde (~), press ~ twice.
