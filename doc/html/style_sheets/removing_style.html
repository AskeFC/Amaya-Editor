Removing Style

HTML provides some character-level elements that allow you to associate style with character strings. You can also remove any style from a character string once it has been associated with a character string.

To remove style from a character string

  1. Select the character string from within the document from which youwant to remove the style.
  2. Open the Style menu, and then select InformationType.
  3. Select the style command from within the menu that you want to remove. For example, if a character string has the style command emphasis associated with it, re-select emphasis.
