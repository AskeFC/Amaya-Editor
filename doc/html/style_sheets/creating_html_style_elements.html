Creating HTML Style Elements

Two submenus from the Style menu let you add HTML elements to your CSS style sheet: Information Type and Character Elements.

InformationTypes

The Information Type menu from the Style menu opens a submenu with HTML elements that define abstract styles. Three of these styles are also available through buttons: emphasis (), strong () and code ().

To use this command:

  1. Select a character string in your page.
  2. Choose an item from the Information Types menu.
  3. The selected character string displays the chosen style.

Character Elements

The Character Elements command from the Style menu lets you change the style of text. The first entries of this menu should be replaced by CSS commands. They are included only to complete the HTML 3.2 implementation.

To use this command:

  1. Select a character string in your page.
  2. Choose an item from the Character Elements menu.
  3. The selected character string displays the chosen style.
