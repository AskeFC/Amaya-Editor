Adding a New Column to a Table

New columns can be easily added to a table using the Enter key.

To add a new column to a table:

  1. To create a new column to the left of an existing cell, click at the beginning of the cell content. To create a new column to the right of an existing cell, move the cursor to the end of the cell content.

  2. Press the Enter key twice to create a new cell and its corresponding column.
