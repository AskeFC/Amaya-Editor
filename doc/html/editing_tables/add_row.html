Adding a New Row

To create a new row:

  1. To create a new row above an existing row, click at the beginning of the first cell in the row and select the Row entry in the Table sub-menu.
  2. To create a new row below an existing row, move the cursor to the end of the last cell in the row and select the Row entry in the Table sub-menu.

New rows can also be easily added to a table using the Enter key. When a whole row is selected (use F2 on Windows and Esc on Linux for that), pressing the Enter key create a new row after the selected row.
