Using client-side image maps

Amaya allows you to use client-side image maps.

To activate an area in an image map

  • Double-click on the area within the image. The URI associated to this area is then retrieved, and opens in the browser.

To display the existing areas within an image map

  • Views menu, then select Show Map Areas.

    Note:    You can also display existing areas directly from the keyboard, by clicking Alt + V, then Alt + M.
