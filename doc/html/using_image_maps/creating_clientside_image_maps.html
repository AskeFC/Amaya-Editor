Creating client-side image maps

Amaya allows you to use client-side image maps, which allow you to link an image to one or multiple URIs.

To create a new image map

  1. Insert an image into the document, by choosing the Types menu and selecting Image.

    Note:    You can also insert an image by clicking the icon on the toolbar, or directly from the keyboard by pressing Ctrl + t (or Ctrl + i in Unix).

  2. Open the Types menu and then select Area. Then choose Circle, Polygon, or Rectangle, depending on the shape of the area within the image that you want to create. The following table indicates how to define an area depending on the shape you select.

     

    Table 1: Defining shapes

    Shape

    Defining the Area

    Circle

    Click the left mouse button to select the position of one corner of the circle. Keep the button pressed down, move your cursor to the opposite corner, then release the button.

    Polygon

    Click the left mouse button at each point that you want to be a corner of the polygon. Right-click to indicate when you have reached the last point.

    Rectangle

    Click the left mouse button to select the position of one corner of the circle. Keep the button pressed down, move your cursor to the opposite corner, then release the button.
  3. When you are finished defining the area of the image map, a dialog will appear that requires Alternate Text. Alternate Text is required for any graphic or image map.
  4. Click anywhere within the image map area. Another dialog appears, and you can input the URI or target to which you want to link the image. Enter the target of the link associated with the defined area, or Browse for the link.

    Tip:     to see an outline of each existing area in the image map, open the Views menu and then select Show map areas.
