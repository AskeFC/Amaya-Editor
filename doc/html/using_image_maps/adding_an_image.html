Adding an Image

To insert an image, choose Image from the Types menu or click on the Graphics button, or use the keyboard shortcut control-t, control-i in Unix. The resulting dialog lets you enter the file name and alternative text ('ALT' text) of the desired image.

If you wish to provide a long description (for example, if your image presents complex information) you should do so in a separate document and link it by giving the URI as a longdesc. This command is also available in the Attributes menu.

To help find an image file you can enter a filter field, and use the filter button, so the dialog will list only the files that match your pattern.
