Selecting by Structure

The structure view is especially convenient for quickly selecting large elements or groups of elements within a document.

To select an element:

  • Click the blue label that represents the element, or click the name of any of its attributes, or click the blue vertical line associated with that element. In this way, a large element like a complex table or a long list can be selected with a single click.

To select a text character or text string within a structure:

  • Clicking a text character in Structure view selects only that character. To select a text string, click and drag across the desired text.

To select an attribute value:

  • In Structure view, attribute values that can be edited from the keyboard are represented in bold text. Click and drag across the attribute value to select it. You cannot extend this type of text selection outside the attribute value.

To select structure elements using the keyboard:

  • Press the Esc (F2 on Windows) key to select the parent element in the document structure. Press the Esc (F2 on Windows) key several times to select containers (and everything in them) up the structure towards the root.
  • Press Ctrl+- (minus key) to select a child element.
  • Press Ctrl+k to select the next element at the same level (or at the level of the parent element if there is no next/previous child of the current level).
  • Press Control-j to select the previous element at the same level (or at the level of the parent element if there is no previous child of the current level).
