The Transform Command

The Transform command performs complex structure transformations, based on a structure transformation language that specifies how a given structural pattern can be transformed into different structures.

Available Transformations

The available transformations are defined in the file Amaya/amaya/HTML.trans. Please refer to the document "Use of the file HTML.trans" for a description of the language in which the transformation rules are written.

Additional transformations may be specified in the HTML.trans file. Some are provided as examples. For example, nested lists can be transformed into a table and back again.

Using the Transform Command

To use this command, follow the procedure below.

  1. Choose Transform from the Edit menu.

    Amaya displays a menu that lists all the transformations that can be applied to the current selection.

  2. Choose the transformation you want.
