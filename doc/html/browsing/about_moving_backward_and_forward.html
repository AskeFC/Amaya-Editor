About Moving Backward and Forward

Amaya creates a history list to keep track of the pages you’ve viewed in the current session. You can move backward or forward through pages in the history list in several ways:

File menu

Choose Back from the File menu to view the previous page.

Choose Forward from the File menu to view the next page.

Keyboard shortcuts

Unix:

Press Alt-b to view the previous page.

Press Alt-f to view the next page.

Windows:

Press Alt-left-arrow to view the previous page.

Press Alt-right-arrow to view the next page.

Button bar

Click the XX button to view the previous page.

Click the OO button to view the next page.
