Unix Platforms

Under Unix, you can set the paper format (A4 or US), portrait or landscape orientation, the number of pages printed on the same sheet (1, 2, or 4 pages per sheet), and the output (file or printer).

In addition, you can select the manual paper feed, generate a table of contents for the document, generate a table of links which creates a number after each link in the document that refers an entry within the table of links and the corresponding URL; print the document URL on each page, and print the document with or without style sheets.
