Moving graphics

After you have created a new SVG graphic, you can move the graphic somewhere else within the document:

To move the graphic directly using the keyboard

  1. Move the cursor into an area of the graphic
  2. Press the Control key and click the mouse. This will highlight the image map area.
  3. Release the Control key but continue to hold down the mouse button. Use the mouse to move the highlighted graphic to its new location.

To move the graphic by editing the attributes

  1. Select the graphic.
  2. Open the Attributes menu and select the attribute you want to edit.
  3. In the dialog, change the attribute, click Apply, then click Close.
