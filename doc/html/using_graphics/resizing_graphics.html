Resizing graphics

After you have created a new SVG graphic, you can resize the graphic.

To resize the graphic directly using the keyboard

  1. Move the cursor into an area of the graphic.
  2. Press the Control key and right-click the mouse. This will highlight the image map area.
  3. Release the Control key but continue to hold down the right mouse button. Use the mouse to change the size of the highlighted graphic.

To resize the graphic by editing the attributes

  1. Select the graphic.
  2. Open the Attributes menu and select the attribute you want to edit.
  3. In the dialog, change the attribute, click Apply, then click Close.
